# PacknMove - Landing Page

A responsive landing page for a packers and movers business built with HTML, Tailwind CSS, and vanilla JavaScript.

## Live Demo

[Live Demo](https://packnmove.vercel.app/)

## About

PacknMove is a fictional packers and movers service landing page built as part of a frontend assignment. It includes a hero section, services overview, contact form, and footer — all fully responsive across mobile and desktop.

## Features

- Responsive navbar with hamburger menu for mobile
- Hero section with stats (customers, experience, rating)
- 3 service cards with hover animations
- Contact form with name, email, phone, service selection, and message
- Smooth scroll navigation — all buttons link to relevant sections
- Scroll-reveal animations on cards and form
- SEO-friendly meta tags and semantic structure

## Tech Stack

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Font Awesome icons
- Google Fonts (Playfair Display)

## Getting Started

1. Clone the repository
   ```
   git clone https://github.com/akisha-dev/PacknMove
   ```

2. Open the project folder
   ```
   cd packnmove
   ```

3. Open `index.html` in your browser — no build step needed

## Deployment

This is a static site with no dependencies to install. You can deploy it directly on:

- **GitHub Pages** — push to a repo and enable Pages in settings
- **Vercel** — drag and drop the folder or connect your GitHub repo
- **Netlify** — same as Vercel
